## 프로젝트 개요

**Enterprise AI Hub**는 대기업 업무 자동화를 목표로 한 AI 기반 사내 포털 시스템입니다.  
각 부서(HR·재무·CS·법무·R&D)의 반복 업무를 OpenAI API(gpt-4o-mini)와 연동하여 자동화합니다.

**Tech Stack**
- Frontend: Vite + React (Functional) + TailwindCSS + Recharts
- Backend: FastAPI (Python) + pg8000 (PostgreSQL)
- AI: OpenAI API (gpt-4o-mini)
- 인증: localStorage 세션 기반 (사번+부서+직급)

### 지시 #023 — 법무 챗봇 진짜 RAG 업그레이드 (벡터 임베딩)
```
현재 법무 챗봇은 키워드 빈도수 기반 검색이야.
OpenAI text-embedding-3-small로 진짜 벡터 RAG를 구현해줘.
pgvector 없이 PostgreSQL REAL[] + numpy 코사인 유사도로 구현해줘.

업로드 시: 청크 분할 → 배치 임베딩 → legal_document_chunks 테이블 저장
질문 시: 질문 임베딩 → 코사인 유사도 → 상위 5개 청크 → GPT 답변
```

**결과물**
- `rag_utils.py` — `embed_text()`, `embed_texts_batch()`, `cosine_similarity()`, `select_top_chunks_by_vector()` 추가
- `legal_chat_service.py` — 전면 교체: `legal_document_chunks` 테이블 (REAL[] 임베딩 컬럼), 배치 임베딩 저장, 벡터 유사도 검색
- `requirements.txt` — `numpy>=2.0.0` 추가
- 문서 목록에 **청크 수** 표시, 답변에 **출처 violet 태그** 표시

**핵심 포인트**
- pgvector DB 확장 불필요 — Python/numpy 레벨에서 코사인 유사도 계산
- 배치 임베딩으로 OpenAI API 호출 최소화 (`batch_size=100`)

## 바이브코딩 패턴 분석

이 프로젝트에서 반복적으로 사용한 AI 지시 패턴입니다.

### 패턴 1 — "역할 부여 후 요구사항 도출"
```
대기업의 [부서명] 실무자라고 생각하고, AIFDE에게 어떤 업무 자동화를 부탁할 것 같아?
```
→ AI가 실무 관점으로 기능 목록을 제안하게 한 뒤, 선별해서 구현 지시

### 패턴 2 — "스키마 기준 명시"
```
데이터는 첨부된 finance_schema.sql에 정의된 컬럼명을 정확히 사용해줘.
(total_amount, receipt_date, account_code 등)
```
→ DB 스키마를 먼저 확정하고 그것을 기준으로 프론트·백엔드를 동시에 지시

### 패턴 3 — "번호 매긴 요구사항"
```
1. 접근 권한: ...
2. 백엔드 API: ...
3. 테스트 데이터: ...
4. 프론트엔드: ...
5. AI 연동: ...
```
→ 한 번의 지시에 5개 레이어를 동시에 요청해서 컨텍스트 일관성 유지

### 패턴 4 — "Human-in-the-Loop 명시"
```
AI 분석 결과를 사람이 검토하고 확정하는 구조로 만들어줘.
```
→ AI 자동화 + 사람 검증 단계를 항상 포함시켜 실무 신뢰성 확보

### 패턴 5 — "권한 계층 명시"
```
재무팀 소속 직원만 들어올 수 있게 해줘.
프론트 라우터 가드 + 서버 사이드 검증 둘 다 해줘.
```
→ 클라이언트·서버 이중 보안을 명시적으로 요청

## 핵심 설계 원칙

1. **API 중앙화** — 모든 fetch는 `frontend/src/api/*.js`에서만 호출
2. **Loading/Error/Empty 상태** — 모든 비동기 UI에 3가지 상태 필수 구현
3. **Mobile-First** — 버튼 최소 44px, 반응형 grid
4. **보안** — `.env`로 시크릿 관리, 하드코딩 금지
5. **한글 주석** — 모든 코드 주석은 한국어
6. **Human-in-the-Loop** — AI 분석 후 사람이 검토·확정하는 단계 포함
